# Git Convention
- **main**: 초기 설정
  - **develop**: 개발 브랜치
    - **feature**: 기능 개발 브랜치
  - **hotfix**: 급한 버그 수정 브랜치
  - **stage**: 스테이징 브랜치
  - **deploy**: 배포 브랜치
---
# Commit Convention
- **feat**: 새로운 기능 추가
- **refactor**: 코드 리팩토링
- **fix**: 버그 수정
- **docs**: 문서 수정
- **env**: 환경 설정 파일 수정
